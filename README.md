# hackintosh-Huawei-D14-2019-AMD
OpenCore EFI for Huawei D14 2019 with AMD Ryzen 5 2500U Processor and integrated Radeon Vega 8 Graphics Card.
Specification:

    Model : 
		Huawei D14 2019 Ryzen
    BIOS : 
		AMI BIOS v1.22
    CPU : 
		AMD Ryzen 5 2500U
    RAM : 
		8GB
    Storage : 
		Lexar NM620 NVMe 1TB
    iGPU : 
		AMD Radeon Vega 8 iGPU
    Wifi/BT : 
		Intel 8265 WIFI + BT
    Ethernet : 
		None
    Audio : 
		Realtek + HDMI Audio
    Boot Mode : 
		UEFI (CSM disabled)
    Bootloader : 
		OpenCore 0.9.7
    OS : macOS Ventura 
		

Whats Work?
  	QE/CI Graphics of Vega8 + DRM
	Wifi/BT
	All USB Ports
	Shutdown and Restart


Whats don't work?
	

Not tested

	CPU Power Management
	Ethernet
	HDMI
	TRIM Support for SSD
